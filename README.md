# Travello - Agentic Travel Planning System

Travello is a modern, collaborative travel planning application designed to make organizing group trips seamless and intelligent. Featuring an Agentic AI travel assistant powered by Mistral AI, Travello helps users discover destinations, plan itineraries, and track budgets in real-time.

## 🚀 Features

- **Collaborative Trip Workspace**: Central hub for planning trips with friends and family.
- **Agentic AI Assistant**: Intelligent chat companion to suggest itineraries and activities using Mistral AI.
- **Destination Discovery**: Search for attractions and seamlessly add them to your travel plan.
- **Itinerary Planning**: Organize daily schedules with nested activities and cost tracking.
- **Budget Tracking**: Real-time spending summaries to keep trips within financial goals.
- **Secure Authentication**: Robust user registration and login system.

## 🛠️ Technology Stack

- **Frontend**: React (Vite), Tailwind CSS v4, Lucide React, Axios.
- **Backend**: Node.js, Express.js.
- **Database**: PostgreSQL with Prisma ORM.
- **AI Integration**: Mistral AI SDK.
