# Travello Client

The modern React frontend for the Travello Travel Planning System.

## 🚀 Setup Instructions

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Configure Environment**:
    - Copy `.env.example` to `.env`.
    - Ensure `VITE_API_URL` points to your running server (default: `http://localhost:5000/api`).

3.  **Start Development Server**:
    ```bash
    npm run dev
    ```

## 🎨 Design System

- Built with **Tailwind CSS v4** for cutting-edge styling.
- Uses **Lucide React** for premium iconography.
- Responsive design for planning trips on the go.

## 🧠 Features

- Markdown-rendered AI chat bubbled.
- Dynamic itinerary and destination search.
- Real-time budget calculation visualization.
